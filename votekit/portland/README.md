## Details/History.

Portland City Council ran its first STV election ever in 2024. The City was split into 4 districts, each filling 3 vacancies.

## Data Source.

The four district CVRs are also available in the
[Harvard Dataverse dataset](https://doi.org/10.7910/DVN/STVUET). These copies
originated from prior work with the MGGG lab.

## Cleaning.

These files were cleaned before this repository was assembled. Undervote,
write-in, and overvote markers were removed, but the exact historical cleaning
procedure was not retained. Researchers requiring exact reproduction of the
official tabulation should compare these profiles against the original CVRs.
