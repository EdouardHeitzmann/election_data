# Data License

The 2009-2021 data in this directory was derived from Sabrina Laverty and
FairVote's
["Proportional ranked choice voting CVRs"](https://doi.org/10.7910/DVN/STVUET)
dataset, released under
[CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/).

The source data was cleaned, reformatted, and supplemented with metadata for
this repository.

The 2025 files were obtained directly from the City of Minneapolis. The City's
[Open Data Policy](https://www.minneapolismn.gov/government/charter-and-code-of-ordinances/city-policies/open-data-policy/)
states that open data should be freely available for use and reuse, but does
not identify a standard Creative Commons or SPDX license for these files.
