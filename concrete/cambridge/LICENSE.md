# Data License

The data in this directory was derived from Sabrina Laverty and FairVote's
["Proportional ranked choice voting CVRs"](https://doi.org/10.7910/DVN/STVUET)
dataset, released under
[CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/).

The source data was cleaned, reformatted, supplemented with metadata, and
converted to ConcreteSTV's JSON `.stv` format for this repository.
